# Filmo

Electron app to view NFO files generated by:
- https://www.tinymediamanager.org/
- https://kodi.tv/
![Cature filmo](https://github.com/radiium/filmo/blob/master/capture1.png?raw=true)




### Prerequisite
node >= v12

### Dev

```bash
npm i
npm start

```

### release

```bash
npm i
npm start
```
